## Project: Thooya Bhoomi
A precision agriculture system focused on soil health monitoring and automated reporting.

### Implementation
* **Hardware:** ESP32, NRF24L01 transceivers.
* **Sensors:** NPK (Nitrogen, Phosphorus, Potassium) soil sensors.
* **Data Flow:** Local sensor grid → Gateway → PHP Backend → MySQL Database.

### Key Outcome
Real-time dashboard visualization allowing farmers to optimize fertilizer usage based on data, not estimation.
